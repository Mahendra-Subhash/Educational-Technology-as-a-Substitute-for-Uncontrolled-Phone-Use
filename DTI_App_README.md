# DTI Problem Statement — EdTech vs Phone Distraction

> **"Educational Technology can substitute uncontrolled phone use with engaging curriculum & interactive tools, keeping students focused."**

A fully responsive, single-page web app presenting the DTI Problem Statement on Educational Technology as a solution to uncontrolled smartphone use in classrooms.

---

## 🚀 Live Preview

Simply open `index.html` in any modern browser — no build step required.

## 🎨 Features

- **8 fully content-rich sections** covering the complete DTI problem statement
- **Animated stat counters** that trigger on scroll
- **Scroll-reveal animations** on every card and content block
- **Sticky navigation** with active section highlighting
- **Reading progress bar** at the top of the page
- **Back-to-top button** that appears on scroll
- **Fully responsive** — works on mobile, tablet, and desktop
- **Zero dependencies** — pure HTML, CSS, and vanilla JavaScript

## 🎓 About

This project was created as part of a **DTI (Design Thinking for Innovation)** college submission exploring how Educational Technology can address the growing problem of uncontrolled smartphone use in modern classrooms.

**Key Argument:** EdTech uses the same psychological engagement hooks as social media — instant rewards, competition, social interaction, and visual stimulation — but channels them into curriculum-aligned learning outcomes.

---

## 📊 Key Statistics Covered

- **72%** of students check phones during class
- **40%** drop in focus and retention after phone use  
- **2.5×** improvement in retention with EdTech
- **89%** of teachers confirm EdTech reduces distraction
- **76%** of students report higher engagement with EdTech
